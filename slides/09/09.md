# Highway Networks

For input $→x$, fully connected layer computes
$$→y ← H(→x, ⇉W\_H).$$

--

Highway networks add residual connection with gating:
$$→y ← H(→x, ⇉W\_H) \cdot T(→x, ⇉W\_T) + →x \cdot (1 - T(→x, ⇉W\_T)).$$

--

Usually, the gating is defined as
$$T(→x, ⇉W\_T) ← σ(⇉W\_T →x + →b\_T).$$
